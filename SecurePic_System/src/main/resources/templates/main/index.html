SecurePic
프로젝트 제안 배경 이미지의 사용 빈도가 급증하면서 이미지 파일 내에 포함된
개인정보와 기밀 정보의 노출 위험성 또한 증가하고 있습니다.
이러한 상황에서 해당 프로젝트 목표는 이미지 분석을 통해
민감한 정보 및 기밀 정보를 효과적으로 탐지하고, 필요한
보안 조치를 제안하는 시스템을 설계 및 구현하는 데 있습니다.
이미지 내 개인정보 탐지 후 삭제 이미지를 업로드하면 개인정보를 탐지합니다.
블러 처리 예상지를 이유와 함께 제공하여 필요한 보안 조치를
안내합니다. 사용자가 블러 처리 예상지 내에서 블러 처리할 곳을
선택하면 선택된 곳만 블러 처리한 사진을 반환합니다. 이미지 내 보안 지역에 대한 정보 탐지 후 삭제 이미지를 업로드하면 보안 지역에 대한 정보를 탐지합니다.
군부대, 공공 기관 등 출입 금지 장소의 정보가
탐지되면 블러 처리 선택 없이 경고 문구를 생성합니다.
이 기능은 이미지가 위치 정보를 포함할 경우에만 제공됩니다.
이미지 내 특정 정보를 탐지하여 삭제하는 시스템 PNG, JPG, JPEG 형식의 이미지를 업로드해주세요.
이미지 업로드
이미지 분석을 통해 개인정보 및 기밀 정보를 탐지 후 삭제하고, 필요한 보안 조치를 제안합니다.
하단에 프로젝트 제안 배경과 더 자세한 기능이 설명되어 있습니다.
